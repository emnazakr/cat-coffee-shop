Cat Coffee Shop (Neko Brew);

Project Description:
Cat Coffee Shop(Neko Brew) is a simple web application built using php andMySQL
The system allows managing coffee shop menu items with full CRUD operations and displays them in a user friendly interface

Technologies Used:
- PHP(MVC)
- MySQL
- HTML5
- CSS3
- JavaScript
- WampServer
- Git & GitHub

Features:
CRUD;
- View menu items
- Add new menu items
- Edit existing items
- Delete menu items
- Categories (Coffee / Desserts)
- Responsive and user-friendly UI
